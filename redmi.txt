-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
==> STEP : 1 
- clone the project. (git clone https://gitlab.com/django_warriors/starter_project_with_custom_user.git)
- create env. (python3.10 -m venv env_starter)
- start environment. ( source env_starter/bin/activate)
- cd starter_project_with_custom_user
- pip install -r requirements/local.txt

## Creating Apps:
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
i) First Create a directory inside apps. Directory name will be same as app name. ex: apps/users.

ii) - python manage.py startapp users apps/users.

iii) Register the app in settings.py. ex:
INSTALLED_APPS = [
    ... other apps,
    apps.example_app.
]

iv) Then go to users --> apps.py --> name = 'apps.users'

#### MakeMigrations and Migrations:

i) python manage.py makemigrations users.
ii) python manage.py migrate users.
